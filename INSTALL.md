# Windsurf 批量注册工具 — 安装说明

## 第一步：解压

将 `windsurf-batch-register-v1.0.0.zip` 解压到任意文件夹，例如桌面。

解压后你会得到这些文件：

```
windsurf-batch-register/
├── manifest.json
├── popup.html
├── popup.css
├── popup.js
├── background.js
├── content-script.js
└── icons/
```

## 第二步：打开 Chrome 扩展管理页面

在 Chrome 浏览器地址栏输入：

```
chrome://extensions/
```

按回车打开。

## 第三步：开启开发者模式

在页面**右上角**，打开 **「开发者模式」** 开关。

## 第四步：加载扩展

点击左上角的 **「加载已解压的扩展程序」** 按钮，选择刚才解压出来的文件夹。

## 第五步：使用

1. 点击浏览器右上角工具栏的扩展图标（拼图图标）
2. 找到 **「Windsurf 批量注册工具」**，点击打开
3. 在文本框中输入账号密码，每行一个，格式：`邮箱  密码`
4. 点击 **🚀 开始批量注册**

## 注意

- 人机验证（Cloudflare）需要**手动点击**复选框
- 邮箱验证码需要**手动输入**
- 注册成功的账号会显示在底部，可一键复制
